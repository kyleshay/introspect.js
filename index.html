View source and in the console inspect the object `adder` and the `Reflect` object

The Reflect object is created with 'adder.js' as the parameter. It loads that file via ajax, introspects the file and injects a property called _privates. This object contains all of the private functions from the original file.
  • With `adder`, you may try adder.do(1, 3) in the console.
  • With the `Reflect` object, you may access the private function doAdd via the following:
    • Reflect.injected._privates["doAdd"](1, 3)
    • Reflect.injected._privates.doAdd(1, 3)
